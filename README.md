# Kurulum kılavuzu  
1. zip dosyasını indirin
   - sağ üstteki yeşil "Code" butonuna basın
     - "Download ZIP" e basın

2. zip dosyasını ayıklayın(içerideki klasörü dışarı alın)

3. Game.sln'i çalıştırın  

4. f5'e basın
